# 📝 To-Do List Web App

A simple, lightweight To-Do List application built using HTML, CSS, and JavaScript. It allows users to:

- ✅ Add tasks
- ✅ Mark tasks as completed (with strike-through effect)
- ✅ Delete tasks
- ✅ Persist tasks using LocalStorage

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API
